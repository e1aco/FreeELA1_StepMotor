#### 修改硬件部分

##### 修改细节：

修改这些主要是因为原本买了些之前项目的材料，懒得再根据新项目买新的了（不想让本不富裕的家庭雪上加霜）

1. 修改LDO LP2992IM5-3.3为TPRT9013-33GB
2. 修改单片机型号为STM32F103RCT6
3. 晶振改为8Mhz的无源晶振
4. 取消SS54的二极管
5. 把串口和烧录口分开来，然后改成2.54的（原本是1.25间距的，我认为不常用）
6. CAN芯片改为SN65HVD230QDR
7. 修改5819封装为SOD-123（这真不是我扣，电源部分我感觉原来的SOD323封装小了）
8. 修改按键型号为SMD,6.1x3.6mm规格的。（排版完以后发现板子还有很多空位，就换成常用的RESET按键了）
9. 改成两层板（画着画着感觉完全可以用两层板，就直接改成两层的了，虽然对于嘉立创来说都可以打，但是正常生产的话两层板确实会便宜不少）
10. 增加485（想顺便学习一下modbus）

##### 可能存在的问题

1. R12和R13到TB67H450FNG,EL芯片之间的过孔可能太少了，由于我排板的时候没留多少空间所以那个过孔我改成了焊盘，要多上点锡
